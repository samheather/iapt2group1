Instructions

Provide basic project information on this page. On the next page, you can add images and specify exactly what fields of information you'd like transcribing.

Title: Provide the title of your project. This should be a short description of your project.

Requested Transcription: Explain to transcribers what information you'd like extracting from your uploaded material. This will be shown with all images when they are transcribed. You will be able to specify on the next page semantic units to be transcribed separately.

Note that once you set your project title and request description you will no longer be able to change them. Think carefully before you submit this page.
